# medRAG
RAG FOR MEDICAL DATA
The repository contains the running codes for Medical rag inside MedRAG folder
1) Content : folder has Fine tuned Model of "DeepSeek-R1-Distill-Qwen-1.5B"
2) Finetune.ipynb : Code used for fine tuning
3) Med_vectordb.ipynb : Code for Vector Db creation
4) rag_pipeline.py : RAG implementation code
5) app.py : Fast API code with HTML structure
